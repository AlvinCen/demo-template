Design tokens that scale

Posted on Sep 23, 2025 · 5 min read

Design tokens are the single source of truth for your UI decisions. With CSS custom properties, you can theme components and pages with ease.

“If you can't name it, you can't change it.” — tokens give names to design decisions.

Start with a palette

Define your brand color, surface, text, border, and radii values. Keep a dark and light mode in mind.

:root { --brand: #2563eb; --text:#0f172a; }

Then, reference these tokens in your utilities and components so style changes cascade everywhere.
