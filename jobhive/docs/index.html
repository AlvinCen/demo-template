JobHive – Job Board & Recruitment HTML Template

Exclusive item for ThemeForest. Version 1.0.0

Structure

|- assets/css/style.css
|- assets/js/main.js
|- assets/img/placeholder/*.svg

|- docs/index.html, docs/license.txt

Features

  • Jobs list with filters (q, location, category, type, experience, salary)
  • Job detail + Apply form (stores to localStorage as applications)
  • Saved jobs (localStorage)
  • Post a Job (demo) → appears in Jobs
  • Employer dashboard (shows recent jobs + application counts)
  • Candidate dashboard (profile + applications filtered by email)
  • Pricing & checkout (demo), Blog, Auth pages
  • Dark mode, responsive, no jQuery, local SVG placeholders only
